Über mich

Ich bin Frontend Softwareentwickler mit abgeschlossenem Studium der Betriebswirtschaftslehre (B.A.) und einer Weiterbildung zum Software Developer (IHK). Mein Schwerpunkt liegt auf der Entwicklung moderner, responsiver Webanwendungen mit HTML, CSS, JavaScript, TypeScript und React.

Technischer Fokus

  • Frontend-Entwicklung mit HTML, CSS, JavaScript, TypeScript und React
  • Responsives Design und klare Benutzeroberflächen
  • Grundkenntnisse im Backend mit Python, FastAPI und SQL
  • Versionskontrolle und Projektarbeit mit Git und GitHub

Hintergrund

Meine Kenntnisse habe ich durch praxisorientierte Projekte, intensives Selbststudium sowie durch die Entwicklung einer Fullstack-Anwendung im Rahmen meiner Abschlussarbeit aufgebaut. Dabei lege ich Wert auf saubere Strukturen, nachvollziehbaren Code und eine gute Nutzerführung.

Arbeitsweise
